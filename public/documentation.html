APP TITLE 'MyFLIX'

* The web application will provide users with access to information about different movies, directors, and genres.

* Users will be able to sign up, update their personal information, and create a list of their favorite movies.

REQUEST URL Http Method Response
Return a list of ALL movies to the user /movies GET returns json objects of all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET returns json object of selected movie title
Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/:name GET returns json objects of all movies included in selected genre
Return data about a director (bio, birth year, death year) by name /directors/:name GET returns json array containing bio/dob/dod keys and values
Allow new users to register /users POST adds a new user to register
Allow users to update their user info (username) /users/:name PUT updates user data
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /favourites/:name PUT adds selected movie title to favourites list
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /favourites/:name DELETE removes selected movie from favourites list
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/:name DELETE deletes user from register
